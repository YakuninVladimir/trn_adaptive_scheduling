\Abstract{
В работе рассматривается задача выбора числа внутренних вычислительных шагов в малой рекурсивной
языковой модели~\cite{trm}. Основная прикладная задача состоит в предсказании следующей языковой
единицы по уже наблюденному префиксу. Число внутренних уточнений не задаётся заранее одинаково для
всех позиций, а рассматривается как объект статистического моделирования. Для каждой позиции
текста по последовательности потерь определяется время остановки, минимизирующее сумму
ошибки прогноза и штрафа за вычисления. Основная модель оценивает условное распределение этого
оракульного времени по информации, доступной на текущем внутреннем шаге. Над предсказанным
распределением строятся различные правила остановки. Описаны и исследованы
 шесть параметрических семейств для распределения времени
остановки, пять вариантов правил остановки, а также схема переиспользования малых языковых моделей. Приведены
результаты на WikiText-103 и на контрольной выборке ARC-AGI после перебора по решётке.
}

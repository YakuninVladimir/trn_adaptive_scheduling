# Вероятностное моделирование времени остановки

Основной файл: `thesis.tex`.

Рекомендуемая сборка в Overleaf:

1. загрузить весь архив проекта;
2. выбрать компилятор `XeLaTeX`;
3. указать `thesis.tex` как главный файл;
4. собрать два раза для обновления содержания и ссылок.

Текущая версия переписана как связный текст диплома для кафедры математической статистики. Основная задача: предсказание условного распределения оракульного времени остановки при рекурсивном прогнозировании речи. Правило остановки рассматривается как отдельная надстройка над этим распределением.

В PDF уже включены титульный лист, аннотация, содержание, введение, постановка задачи, вероятностная модель речи, побочная модель ARC-AGI, описание рекурсивной модели, сравнение с низкоранговой донастройкой LoRA, семейства распределений, пять правил остановки, обучение, план экспериментов, шаблоны графиков и таблиц, заключение и приложения.
